■使用言語
　Python 3.11.1

■作成ファイル
　１）program1.py：課題１
　２）program2.py：課題２

■課題内容
　課題１：
　指定された金額に対して、日本の硬貨で最小何枚になるかを計算するプログラムを書いてください。
　　例：　2439円 であれば、16枚 が答え
　 　　(500円：4枚　100円：4枚　10円：3枚　5円：1枚　1円：4枚 なので、16枚)
　　　　　1467円 であれば、11枚 が答え

　課題２：
　　課題１に条件を追加し、500円：8枚、100円:5枚、50円：１枚、10円：4枚、１円：9枚 を
　　所持している場合で、課題１と同じように指定された金額に対して、枚数を答えるプログラムを
　　書いてください。

■こだわりポイント
　・金額入力の際に整数値であるかのバリテーションチェックを付けた。
　・整数値でない場合、5回まで再入力可能にした。
　・使用する硬貨をリストにすることで、硬貨の金額や枚数変更に対応しやすくした。
　・課題２について、硬貨が足りない場合のエラー処理を追加した。
